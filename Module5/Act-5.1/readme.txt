
 * installer les dépendances :
	- composer install
 
 * création de la base de données:
	- php bin/console doctrine:database:create

 * importer le fichier db_security.sql dans la base précédemment créé.

 * Lancer le serveur:
	 - php bin/console server:run

 * coordonnées d'un compte admin:
	adresse email: admin@hotmail.com
	mot de passe : 123

 * page accessible par un compte avec un role user : http://127.0.0.1:8000/user

 * page accessible par un compte avec un role admin : http://127.0.0.1:8000/admin

 * page se differencie selon le role : http://127.0.0.1:8000/
